			이벤트페이지(당첨자발표등), FAQ, 매거진(주제(날씨, 계절, 지역 축제연관 등..)별 맛집관련 기사 연재..)

------------기능---------------

1. 확실한 검색 위치 설정 
	-> 맛집이 소재하는 위치를 반드시 검색창으로 검색 혹은 키워드 검색 후 
	   지도상에서 세부 위치 설정 가능하게 할 것.(선택사항, 클릭하면 지도가 나옴, 기준위치의 마커를 이동
	   하여 새로운 좌표를 찍으면, 해당 좌표기준으로 상위권 맛집들을 마커로 보여줌.(좌표정보필요))
	   	
		=>  구글/네이버/다음 지도 api를 이용.

2. 키워드 검색 -> 키워드는 대부분 위치/분위기/가격/특별한 날(기념일,공휴일)/
   같이 먹을 상대(혼밥/동료/직장/데이트)/진짜 맛집 등으로 이루어져 있음. 
   이 중에 진짜 맛집/실패 하지 않을/최고 맛집 같은 키워드는 리뷰 점수 중에서 
   일정 리뷰 숫자 이상을 가지고 있는 데이터를 가져올 수 있게 하는게 좋다고 생각
   (음식점 리뷰는 리뷰점수의 신뢰도점수를 추가
	(ex: 1 ~ 30건 -> 1점, 31 ~ 100건 -> 2점, 101 -> 300건 -> 3점, 301 ~ 1000건 -> 4점, 1001건이상 -> 5점)

3. 많은 사람들이 이미 돈을 받고 쓴 블로그/맛 없는 맛집에 실패하고 있으며 따라서 보다 
   정확한 리뷰를 제공받아 간결하게 이용자에게 제공해야함 -> 메인 화면에 검색창을 넣어서 
   사용자들이 메인 창에서 키워드 입력 -> 엔터 -> 맛집 리스트가 나오도록 제공. 

4. 예산 설정 -> 많은 벤치마킹할 사이트들이 예산을 $,$$,$$$ 로 지정해서 구분하고 있음. 
   6000원 이하, 10000원 이하, 30000원 이하, 그 이상 같이 3~5개로 구분하여 
   사용자들이 예산에 따라 검색할 수 있도록 하는게 좋을 것 같음.

5. 회원 기능 -> 단 한번의 식사를 위해 맛집검색 서비스를 찾는게 아닌 지속적으로 
   비슷한 상황(비슷한 키워드) 에서 검색을 하기 때문에 회원 페이지에서 
   원하는 맛집을 찜하거나 추천 검색어를 제공하는 기능이 있으면 더 좋을듯.
	=> 마이페이지에서 지도api와 연동하여 자신이 찜한 맛집을 한눈에 볼 수 있게 하기.
		-> ex) 서울 선택시 서울에 소재한 맛집들만 지도에 순위와 함께 마킹됨.
		       전국 선택시 전국에 소재한 맛집들이 지도에 순위와 함께 마킹됨.
		(마커에는 리뷰점수와 연동하여 1,2,3,4,5 표시..)

6. 리뷰 기능 -> 로그인을 안해도 되는 1-2줄 정도의 비회원 리뷰(사진 업로드 x) 와 
   반대로 회원리뷰는 10장 이내의 사진을 올릴 수 있게끔 하고 리뷰를 
   올릴 때마다 일정 점수를 주어 회원들 중에서도 경쟁심리로 더욱 많은 리뷰를 남기게끔 유도.
   일정 리뷰점수 이상의 회원은 높은 등급을 부여하여 다른 회원들이 높은 등급의 회원의 리뷰를 
   더욱 신뢰하게끔 함. => 일정 점수가 쌓여 파워리뷰어? 가 되면 리뷰페이지 상단에 위치하도록 하기.
   및 리뷰점수를 많이 획득한 사람은 최대 맛집 등록갯수를 기본100개에서 등급에따라 늘려주기
   , 최대 맛집 등록가능 갯수는 돈주고 사서 영구적으로 늘릴 수 있음.

7. 음식점 상세페이지에 주차가능 여부 및, 부대시설을 보여주는 탭 추가. 탭 (부대시설, 리뷰, 위치 및 소개, 메뉴)

회원vs비회원 기능
===========

필요 기능 -> 
1. 비로그인 회원

<서비스 이용>
a.메인 페이지 검색창에 현재 위치 설정 및 키워드 입력 
b.키워드로 사이트의 db에서 자료를 검색해 와서 창에다가 맛집 리스트 및 지도를 출력. 
c.현재 위치 검색하는 구글 api 에서 gps 정보를 가져와서 일정 거리안에 맛집 리스트를 출력
d.출력된 리스트를 종합점수/위치/총 리뷰 숫자 등으로 정렬 가능한 기능 제공
e.관심 가는 식당을 클릭하면 자세한 정보(위치/리뷰/점수/전화번호/영업시간 등의 자세한 정보) 및 다른 회원/비회원들의 리뷰를 볼 수 있는 기능 제공

<비회원 리뷰 쓰기 제공>
e."비회원 후기" 를 검색된 맛집 카테고리에 작성할 수 있는 기능 (사진 업로드 불가)
f."회원 후기" 를 "비회원 후기" 보다 종합 점수에 가중치를 두어 리뷰를 작성하고 싶은 비회원들이 점차 회원으로 가입하도록 유도 (회원들에게 특정한 
아이디 색깔/이미지 아이콘/레벨별 접근가능 게시판 
설정/마음에 드는 식당 리스트 저장 등의 추가 편의 제공)


B. 로그인 회원

<서비스 이용>
a. 먼저 로그인
b. 서비스 이용 자체는 비회원 서비스와 같이 제공

<회원 리뷰 쓰기 제공>
c. "회원 후기"를 검색된 맛집 카테고리에 작성할 수 있는 기능 (사진 업로드 가능)
d. 회원이 작성한 "회원 후기"의 숫자를 나타내어 얼마나 신뢰가능한 회원인지 다른 회원들이 볼 수 있는 기능 (더 많은 사용자들이 회원 후기 점수를
 올리려는 인센티브 -> 더 많은 신뢰가는 후기 -> 더 많은 사용자)
e. 추가 회원 서비스 (특별 게시판/favourite 리스트/차별화된 아이디 표시/쿠폰 및 타 서비스 제공)
